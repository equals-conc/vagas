### Vagas na Equals

Espaço para a divulgação de vagas para desenvolvedores via **issues** do Github.

Vagas disponíveis: https://github.com/equals-conc/vagas/issues

### Cadastrando uma vaga

Abra uma **issue** no repositório com o título:

```javascript
issueTitle = '[' + city + '] ' + jobTitle 
```

Exemplo: `[Lavras] Desenvolvedor Javascript`

Crie uma vaga [aqui](https://github.com/equals-conc/vagas/issues/new)!

Adicione **labels** contendo o nível de experiência desejada e a forma de contratação.

### Importante

Assim que uma vaga for preenchia, favor fechar a sua issue.
